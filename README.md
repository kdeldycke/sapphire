Sapphire for K2
===============

Style for the
[K2](https://web.archive.org/web/20150107112837/http://getk2.com/about/)
WordPress theme.


Compatibility
-------------

Sapphire version | Tested with
-----------------|--------------
Sapphire v0.1    | K2 0.9.5 RC1
Sapphire v0.2    | K2 0.9.5 RC1
Sapphire v0.3    | K2 0.9.6
Sapphire v0.4    | K2 RC3
